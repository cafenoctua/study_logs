# Rust 学習リポジトリ

このリポジトリは各種技術の学習用です。Rust 学習は `rust_intro/` ディレクトリで行っており、
Claude Code は **Learning モード** (`outputStyle: Learning`) で動作しています。

## Learning モードのルール

**ファイルを編集する前に必ずユーザーへ確認を求めること。**
- ユーザーからの明示的な承認なしに、いかなるファイルも直接編集してはならない。
- 変更提案は diff 形式または説明文として先に提示し、「この変更を適用しますか？」と確認してから実行する。
- コード提案はコードブロックで示し、ユーザーが「適用して」「やって」と明示的に指示するまで待つ。

## プロジェクト構成（Rust 関連）

```
rust_intro/
├── Rust.md              # メインの学習ノート
├── self_made_project.md # 自作プロジェクト一覧
└── projects/            # 実習プロジェクト群 (Rust 2024 edition)
.claude/
├── skills/              # カスタムスキル
└── session/             # セッションログ（gitignore 済み）
```

## 利用可能なスキル

| スキル | 説明 |
|--------|------|
| `/rust-docs` | Rust 公式ドキュメントの検索・リンク取得 |
| `/rust-review` | Rustacean 視点でのコードレビュー |
| `/rust-log` | セッションの学習ログを保存 |
| `/rust-summary` | セッションログを集約して復習問題を生成 |
